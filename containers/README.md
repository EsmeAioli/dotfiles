# Containers

This is where I’ll define containerised dev environments for the distros I care about:

- Ubuntu 🐧
- Fedora 🔥
- Bluefin 🌊

Right now, this is just a scaffold — I’ll be fleshing out my Distrobox setups over time.

Each container will be:

- Based on a clear `Containerfile`
- Bootstrapped with a setup script (where needed)
- Used for specific dev contexts (e.g. Go, Python, observability tools)

For now, this section serves to show how I structure and think about reproducible environments.

More to come!
